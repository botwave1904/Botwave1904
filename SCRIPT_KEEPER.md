# Fleet Commander: SCRIPT KEEPER Methodology

## Core Principles
- Offline-First Intelligence
- Minimal External Dependencies
- Local Processing Priority
- Adaptive, Self-Organizing System

## Architecture Pipeline
```
ingest.py → brain.py → execute.py → reply.py
```

## Technology Stack
- Language: Python, Rust
- Frontend: React Native
- Backend: FastAPI
- ML: Scikit-learn (Embedded)
- Database: SQLite
- State Management: Custom Binary Protocol

## Development Philosophy
1. Maximum Local Processing
2. Zero Cloud Dependency
3. Encrypted Offline Repositories
4. Event-Driven Microservices

## Agent Coordination
- Asynchronous Messaging
- Explicit State Transfer
- Minimal Communication Overhead

## Performance Targets
- Sub-100ms Response Times
- &lt;50MB Total Footprint
- Offline-Capable
- Cross-Platform Compatibility

## Security Principles
- Local-First Data Storage
- Minimal Attack Surface
- No External API Calls
- Encrypted Communication Channels
